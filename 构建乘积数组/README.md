## 构建乘积数组

### 矩阵图说明

| B矩阵 |     |       |      |      |      |      |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
|  B0  |  1   |  A1  |  A2  |  ... | An-2 | An-1 |
|  B1  |  A0  |  1   |  A2  |  ... | An-2 | An-1 |
|  B2  |  A0  |  A1  |  1   |  ... | An-2 | An-1 |
|  ... | ...  | ...  |  ... |  ... |  ... |  ... |
|  ... | ...  |  ... |  ... |  ... |  ... | ...  |
| Bn-2 |  A0  |  A2  | A3   |  ... |  1   | An-1 |
| Bn-1 |  A0  |  A1  |  A2  |  ... | An-2 |  1   |

其中Bi是每一行的乘积，于是矩阵分为了下三角和上三角。

### 思路

- 对于下三角 B'[i+1] = B'[i] * A[i-1]
- 对于上三角 B''[i-1] = B''[i] * A[i-1]
- 再将B'与B''相乘就得到B
